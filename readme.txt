玩法：
点击debug文件夹中的exe运行。
打开exe，经过一系列动画之后，目录出现（可以用鼠标滚轮滑动）。
绿色：已解锁，未通过。
蓝色：已通过。
灰色：未解锁。
点击相应按钮，进入关卡。每通过一个关卡，解锁新关卡。
目标：
从S走到T，途中遇到黑点要转弯，遇到白点要直行。
要经过所有的点，但不一定需要经过所有格子。
W,S,A,D或方向键控制路线。Backspace撤销。
鼠标点击Back，返回目录。
鼠标左键可以打草稿，鼠标右键可以擦除草稿。
鼠标点击Clear按钮，可以清除所有草稿。
一个关可能有多解，但不可能无解O(∩_∩)O
